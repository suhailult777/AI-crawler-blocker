== Installation ==

1. Upload `crawlguard-pro` folder to `/wp-content/plugins/` directory
2. Activate the plugin through 'Plugins' menu in WordPress
3. Navigate to CrawlGuard Pro in your admin menu
4. Follow the setup wizard to configure your settings
